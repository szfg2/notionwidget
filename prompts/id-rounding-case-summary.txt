# Purpose
Summarise an inpatient infectious disease case into a concise, rounding-ready format.

# Instruction
Summarise this infectious disease case into the template below. Be concise but clinically complete. Prioritise the diagnosis, microbiology, source control, antimicrobial plan, duration, pending results, and today's actionable jobs. Do not invent missing details. If information is absent, write "not documented" or "pending".

# Final output format
Finish with:
1. Problem representation: one sentence.
2. ID impression: diagnosis and likely source.
3. Microbiology: organism, date, susceptibilities, clearance status.
4. Antimicrobial plan: drug, dose, route, duration, stop/review date.
5. Source control: completed vs pending.
6. Today: three to five concrete jobs.
7. Pending: results/imaging/procedures to chase.

# Quality check before using on rounds
- Is there a clear syndrome and source?
- Are culture dates and clearance dates explicit?
- Is source control addressed?
- Is antibiotic duration anchored to a start point?
- Are pending items assigned to someone?
- Is today's plan more useful than "continue antibiotics"?
